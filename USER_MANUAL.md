## Getting Started

### Accessing the Admin Panel

1. Navigate to your website's admin panel (usually at `yourwebsite.com/admin`)
2. Log in with your username and password
3. You'll see the main dashboard with all available collections

### Navigation Overview

The admin panel is organized into several sections:

- **Media**: Upload and manage images, videos, and other files
- **Pages**: Create and edit website pages
- **Posts**: Create and manage blog posts
- **Projects**: Create and manage project portfolios
- **Categories**: Organize posts and projects into categories
- **Users**: Manage user accounts (admin only)

## Managing Media (Photos & Images)

### Uploading Images

1. Click on **Media** in the left sidebar
2. Click the **Create New** button (or drag and drop files)
3. Select an image file from your computer
4. Fill in the following:
   - **Alt Text**: A brief description of the image (important for accessibility and SEO)
   - **Caption**: Optional caption that appears below the image
5. Click **Save**

### Image Best Practices

- **File Formats**: Use JPG for photos, PNG for graphics with transparency
- **File Size**: Keep images under 2MB for faster loading
- **Dimensions**: Recommended minimum width is 1200px for full-width images
- **Alt Text**: Always add descriptive alt text for accessibility

### Image Sizes

The system automatically generates multiple sizes of your images:

- **Thumbnail**: 300px wide
- **Square**: 500x500px
- **Small**: 600px wide
- **Medium**: 900px wide
- **Large**: 1400px wide
- **XLarge**: 1920px wide
- **OG**: 1200x630px (for social media sharing)

### Focal Point

When viewing an image in the media library, you can set a focal point by clicking on the image. This ensures important parts of the image remain visible when it's cropped to different sizes.

## Creating Projects

Projects are used to showcase your portfolio work, case studies, or completed projects.

### Step-by-Step Guide

1. Navigate to **Projects** in the left sidebar
2. Click **Create New**
3. Fill in the required fields:

#### Basic Information

- **Title**: Enter the project title (e.g., "Modern Office Building")
- **Slug**: Automatically generated from the title, but you can customize it (the URL-friendly version)

#### Content Tab

- **Hero Image**: Upload a main image that represents the project (appears at the top of the project page)
- **Content**: Write a detailed description of the project using the rich text editor
  - You can format text (bold, italic, headings)
  - Add links, lists, and other formatting
  - Insert images, code blocks, or media blocks within the content
- **Display Mode**: Choose how the gallery should display
  - **Grid**: Images shown in a grid layout
  - **Carousel**: Images shown in a sliding carousel
- **Image Gallery**: Add multiple images to showcase the project
  - Click **Add Image** for each gallery image
  - Optionally add captions for each image
- **Before Image**: Upload a "before" photo for before/after comparison
- **After Image**: Upload an "after" photo for before/after comparison

#### Meta Tab

- **Related Projects**: Link to other related projects (optional)
- **Categories**: Select one or more categories to organize your project (e.g., "Commercial", "Residential")

#### SEO Tab

- **Meta Title**: The title that appears in search results (defaults to project title)
- **Meta Description**: A brief description for search engines (150-160 characters recommended)
- **Meta Image**: The image that appears when sharing on social media (defaults to hero image)

#### Sidebar Settings

- **Published Date**: Set when the project was completed or published
- **Authors**: Select the author(s) of this project

### Publishing a Project

1. Click **Save Draft** to save your work without publishing
2. Click **Publish** when ready to make the project live
3. Use **Schedule Publish** to automatically publish at a future date/time

### Preview Your Project

Before publishing, you can preview how the project will look:

- Click the **Preview** button in the top right
- This opens your project in a new tab as visitors will see it

## Creating Blog Posts

Blog posts are perfect for news, articles, updates, or any time-based content.

### Step-by-Step Guide

1. Navigate to **Posts** in the left sidebar
2. Click **Create New**
3. Fill in the required fields:

#### Basic Information

- **Title**: Enter the post title
- **Slug**: Automatically generated from the title (customizable)

#### Content Tab

- **Hero Image**: Upload a featured image for the blog post
- **Content**: Write your blog post content using the rich text editor
  - Use headings (H1, H2, H3, H4) to structure your content
  - Format text with bold, italic, and other options
  - Add links, lists, and images
  - Insert code blocks, banners, or media blocks as needed

#### Meta Tab

- **Related Posts**: Link to other related blog posts
- **Categories**: Select categories to organize your post

#### SEO Tab

- **Meta Title**: Search engine title (defaults to post title)
- **Meta Description**: Brief description for search results
- **Meta Image**: Social media sharing image

#### Sidebar Settings

- **Published Date**: When the post was/will be published
- **Authors**: Select the author(s)

### Publishing a Blog Post

1. Save as draft to continue editing later
2. Click **Publish** to make it live immediately
3. Use **Schedule Publish** to publish at a specific date/time

## Creating and Editing Pages

Pages are the main content pages of your website (Home, About, Services, Contact, etc.).

### Creating a New Page

1. Navigate to **Pages** in the left sidebar
2. Click **Create New**
3. Fill in the required fields:

#### Basic Information

- **Title**: The page title (e.g., "About Us", "Our Services")
- **Slug**: The URL path (e.g., "about-us" creates `/about-us`)

#### Hero Tab

The hero section appears at the top of your page. You can choose from different hero styles:

- **None**: No hero section
- **High Impact**: Full-screen hero with large text, images, and USP (Unique Selling Points)
- **Fifty Fifty**: Split layout with content on one side and media on the other
- **Hero Collage**: Hero with three images displayed in a collage layout

Each hero type has different customization options:

- **High Impact**: Add rich text content, media, links/buttons, and USP items with icons
- **Fifty Fifty**: Add rich text content, media, and links/buttons
- **Hero Collage**: Add a title, description, and exactly three collage images

#### Content Tab

This is where you build your page layout using **Content Blocks**. Click **Add Block** to add content sections:

**Available Content Blocks:**

- **Content**: Rich text content with flexible column layouts
- **Media Block**: Display images or videos
- **Call to Action**: Prominent sections with buttons and links
- **Content Section**: Structured content sections
- **Text with Image**: Side-by-side text and image layouts
- **Feature Block**: Highlight key features or benefits
- **Process**: Step-by-step process display
- **FAQ**: Frequently asked questions section
- **Suitable Applications**: Showcase suitable uses/applications
- **Cards Carousel**: Display cards in a carousel
- **Image Carousel**: Image slider/carousel
- **Marquee**: Scrolling text or logos
- **Team Section**: Display team members
- **Reviews**: Customer reviews and testimonials
- **Archive**: Display collections of posts or projects
- **Form**: Contact forms or other forms

You can add multiple blocks, rearrange them by dragging, and delete blocks you don't need.

#### SEO Tab

- **Meta Title**: Page title for search engines
- **Meta Description**: Description for search results
- **Meta Image**: Social sharing image

### Editing an Existing Page

1. Go to **Pages** in the sidebar
2. Click on the page you want to edit
3. Make your changes
4. Click **Save Draft** or **Publish** to update

## Working with Content Blocks

Content blocks are the building blocks of your pages. Here's how to use the most common ones:

### Content Block

Add rich text content with flexible layouts:

1. Click **Add Block** → Select **Content**
2. Add columns (1-4 columns supported)
3. For each column, choose the width:
   - One Third (33%)
   - Half (50%)
   - Two Thirds (67%)
   - Full (100%)
4. Write your content in the rich text editor
5. Optionally enable a link for the column

### Call to Action Block

Create prominent call-to-action sections:

1. Click **Add Block** → Select **Call to Action**
2. Choose the style (Style 1 or Style 2)
3. Add your text content
4. Add one or more buttons/links
5. Choose button appearance (default, outline, or CTA style)

### Text with Image Block

Display text alongside an image:

1. Click **Add Block** → Select **Text with Image**
2. Upload or select an image
3. Add your text content
4. Choose the layout (image left or right)

### Media Block

Display images or videos:

1. Click **Add Block** → Select **Media**
2. Upload or select a media file
3. Optionally add a caption

### Image Carousel Block

Create an image slider:

1. Click **Add Block** → Select **Image Carousel**
2. Add multiple images
3. Optionally add captions for each image

### FAQ Block

Add frequently asked questions:

1. Click **Add Block** → Select **FAQ**
2. Choose a style (Style 1 or Style 2)
3. Add a title for the FAQ section
4. Add multiple question/answer pairs:
   - Question (required)
   - Answer (required)

### Feature Block

Highlight key features:

1. Click **Add Block** → Select **Feature**
2. Add an optional title
3. Optionally add background, left, or right media images
4. Add multiple features, each with:
   - Title (required)
   - Description (required)
   - Icon (choose from available icons)

### Process Block

Display a step-by-step process:

1. Click **Add Block** → Select **Process**
2. Add steps with titles and descriptions
3. Steps are displayed in order

### Cards Carousel Block

Display cards in a sliding carousel:

1. Click **Add Block** → Select **Cards Carousel**
2. Add cards with images, titles, and descriptions
3. Each card can have a link

### Marquee Block

Create a scrolling text or logo display:

1. Click **Add Block** → Select **Marquee**
2. Add items to scroll (text or images)
3. Configure speed and direction

### Team Section Block

Display team members:

1. Click **Add Block** → Select **Team Section**
2. Add team members with photos, names, and roles

### Reviews Block

Show customer reviews:

1. Click **Add Block** → Select **Reviews**
2. Add review cards with ratings, text, and author information

### Archive Block

Display collections of posts or projects:

1. Click **Add Block** → Select **Archive**
2. Choose what to display (Posts or Projects)
3. Configure filters and layout

### Form Block

Add contact forms:

1. Click **Add Block** → Select **Form**
2. Select an existing form or create a new one
3. Forms are managed separately in the Forms collection

## SEO Settings

SEO (Search Engine Optimization) helps your content rank better in search results.

### For Pages, Posts, and Projects

Each content item has an **SEO** tab with:

1. **Meta Title**
   - The title that appears in search results
   - Should be 50-60 characters
   - Defaults to your content title if left empty

2. **Meta Description**
   - A brief description for search results
   - Should be 150-160 characters
   - Write a compelling summary that encourages clicks

3. **Meta Image**
   - The image shown when sharing on social media
   - Recommended size: 1200x630px
   - If not set, the hero image will be used

### SEO Best Practices

- **Unique Titles**: Each page/post should have a unique title
- **Descriptive Descriptions**: Write clear, engaging descriptions
- **Keyword Optimization**: Use relevant keywords naturally
- **Alt Text**: Always add alt text to images
- **Internal Links**: Link to related content on your site

## Publishing Content

### Draft vs. Published

- **Draft**: Content saved but not visible to visitors
- **Published**: Content is live and visible on your website

### Publishing Workflow

1. **Create or Edit**: Make your changes
2. **Save Draft**: Save your work without publishing (you can continue editing)
3. **Preview**: Use the Preview button to see how it will look
4. **Publish**: Click **Publish** to make it live
5. **Schedule Publish**: Set a future date/time to automatically publish

### Scheduled Publishing

You can schedule content to be published automatically:

1. Set the **Published Date** in the sidebar
2. Click **Schedule Publish**
3. The content will automatically go live at the specified time

### Unpublishing

To remove content from your website:

1. Open the published content
2. Change the status to **Draft**
3. Click **Save**

## Managing Categories

Categories help organize your posts and projects.

### Creating a Category

1. Navigate to **Categories** in the left sidebar
2. Click **Create New**
3. Enter a **Title** (e.g., "Commercial Projects", "News")
4. The **Slug** is automatically generated
5. Click **Save**

### Using Categories

When creating or editing posts or projects:

1. Go to the **Meta** tab
2. In the **Categories** field, select one or more categories
3. Content can belong to multiple categories

### Benefits of Categories

- Organize content by topic
- Create category archive pages
- Help visitors find related content
- Improve site navigation

## Tips & Best Practices

### Content Creation

1. **Write Clear Headings**: Use headings to structure your content (H1, H2, H3)
2. **Use Short Paragraphs**: Break up long text for better readability
3. **Add Images**: Visual content makes pages more engaging
4. **Proofread**: Always review your content before publishing

### Image Management

1. **Optimize Images**: Compress images before uploading when possible
2. **Use Descriptive Filenames**: Name files descriptively (e.g., "office-building-exterior.jpg")
3. **Add Alt Text**: Always include alt text for accessibility
4. **Set Focal Points**: Use focal points for important images

### SEO

1. **Fill in Meta Descriptions**: Don't skip the SEO tab
2. **Use Keywords Naturally**: Avoid keyword stuffing
3. **Link Internally**: Link to other relevant pages on your site
4. **Update Content Regularly**: Fresh content helps SEO

### Organization

1. **Use Categories**: Organize content with categories
2. **Tag Related Content**: Link related posts and projects
3. **Keep Content Updated**: Review and update old content periodically

### Publishing

1. **Preview First**: Always preview before publishing
2. **Save Drafts**: Save frequently while working
3. **Schedule Posts**: Use scheduled publishing for timely content
4. **Check Published Date**: Ensure the published date is correct

## Troubleshooting

### Image Not Displaying

- Check that the image is uploaded to the Media library
- Verify the image file is not corrupted
- Ensure the image is published (not in draft)

### Changes Not Appearing

- Make sure you clicked **Publish** (not just Save Draft)
- Clear your browser cache
- Check that the content status is "Published"

### Can't Find Content

- Check the search bar in the collection
- Verify you're looking in the correct collection (Pages, Posts, or Projects)
- Check if content is in draft mode

### Can't Edit Content

- Ensure you're logged in with appropriate permissions
- Check if another user has the content locked
- Refresh the page and try again
